Os textos presentes na pasta "samaritan_locally_exported" são transcrições dos caracteres quadraticos para os caracteres samaritanicos gerados a partir do export_as_samaritan.py.

Os textos presentes na pasta "samaritan_website" são consumos do website "https://samaritantorah.com/", havendo centenas de diferenças em relação a este repositório. O arquivo "comparsion.json" contém a comparação entre as duas fontes de transcrição.

Referências digitalizadas para análise e comparação:
 - https://samaritantorah.com/samaritan-pentateuch-manuscripts
 - https://www.digitalcollections.manchester.ac.uk/view/MS-SAMARITAN-00001/9
 - https://www.digitalcollections.manchester.ac.uk/view/MS-GASTER-SAMARITAN-01133/6
 - https://digitalcollections.nypl.org/items/11c25470-6052-013a-7da7-0242ac110003?canvasIndex=1
